# System Failure Modes (Abstract)

## 1. Interaction-Level Failures
- Misreading user intent  
- Over-reactive adaptation  
- Incorrect pacing selection  

## 2. State-Level Failures
- Drift misclassified as focus  
- Constraint misread as reflection  

## 3. Reasoning Failures
- Over-expansion  
- Under-explanation  
- Looping reasoning  

## 4. Safeguards
- Fallback interaction mode  
- Safety API  
- Response compression mechanism
